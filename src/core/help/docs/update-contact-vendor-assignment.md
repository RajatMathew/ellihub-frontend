# Assign a Contact to Vendor from Contact Tab

Use this workflow when an existing contact needs to be linked to a vendor from the Contacts tab. Updating the association keeps directory records, vendor relationships, and project communication references accurate.

## Find the Contact

Navigate to **Directory > Contacts** and use search to find the contact you need to update.

![Contacts search results](/help/scribe/updating-vendor-assignment-for-a-contact/scribe-assets/step-01.png)

Select the contact from the results.

![Contact selected from search results](/help/scribe/updating-vendor-assignment-for-a-contact/scribe-assets/step-02.png)

## Open the Edit Form

Select **Edit Contact** from the contact detail page.

![Edit Contact action](/help/scribe/updating-vendor-assignment-for-a-contact/scribe-assets/step-03.png)

## Update the Vendor

In the **Association** section, click the **Vendor** search field.

![Vendor search field](/help/scribe/updating-vendor-assignment-for-a-contact/scribe-assets/step-04.png)

Choose the correct vendor from the search results.

![Vendor option selected](/help/scribe/updating-vendor-assignment-for-a-contact/scribe-assets/step-05.png)

## Save the Contact

Select **Save Changes** to apply the new vendor assignment.

![Save Changes action](/help/scribe/updating-vendor-assignment-for-a-contact/scribe-assets/step-06.png)

After the save completes, close the confirmation toast or return to the directory.

![Save confirmation](/help/scribe/updating-vendor-assignment-for-a-contact/scribe-assets/step-07.png)

Use **Team Directory** to return to the broader directory area.

![Team Directory navigation](/help/scribe/updating-vendor-assignment-for-a-contact/scribe-assets/step-08.png)

## Quick Checklist

- The contact record is the correct person.
- The **Vendor** tab is selected in the association section.
- The selected vendor is the company that should own or represent the contact.
- **Save Changes** is selected after the vendor is updated.
