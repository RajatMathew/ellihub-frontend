# Replace the Entity Linked to the Contact

Use this workflow when a contact is linked to the wrong entity and needs to be reassigned to a different one.

## Find the Contact

Navigate to **Directory > Contacts** and search for the contact you need to update.

![Contacts search results](/help/scribe/replace-group-contact-association/scribe-assets/step-01.png)

Select the contact from the results.

![Contact selected from search results](/help/scribe/replace-group-contact-association/scribe-assets/step-02.png)

## Open the Current Association

Select the currently linked entity from the contact detail page.

![Current association selected](/help/scribe/replace-group-contact-association/scribe-assets/step-03.png)

Select **Edit Contact**.

![Edit Contact action](/help/scribe/replace-group-contact-association/scribe-assets/step-04.png)

## Choose the Replacement

In the association section, click **Search GCs...**.

![GC search field](/help/scribe/replace-group-contact-association/scribe-assets/step-05.png)

Type the replacement general contractor name or search term.

Choose the replacement general contractor from the results.

![Replacement GC selected](/help/scribe/replace-group-contact-association/scribe-assets/step-07.png)

Confirm the change by selecting **Replace Association**.

![Replace Association action](/help/scribe/replace-group-contact-association/scribe-assets/step-08.png)

## Save the Contact

Select **Save Changes** to apply the new association.

![Save Changes action](/help/scribe/replace-group-contact-association/scribe-assets/step-09.png)

After saving, close the confirmation toast or review the updated contact record.

![Save confirmation](/help/scribe/replace-group-contact-association/scribe-assets/step-10.png)

## Quick Checklist

- The selected contact is the correct person.
- The existing linked entity is the one that should be replaced.
- The replacement general contractor is selected from the search results.
- **Replace Association** is confirmed before saving.
- **Save Changes** is selected after the replacement.
