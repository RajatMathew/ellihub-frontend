# Create New Vendor

Use this workflow when a new vendor needs to be added to the directory. Vendor profiles store business details, status, tax information, payment terms, and optional linked contacts.

## Start a New Vendor

Navigate to **Directory > Vendors**.

![Vendor directory](/help/scribe/add-new-vendor-to-directory/scribe-assets/step-01.png)

Select **Add Vendor**.

![Add Vendor action](/help/scribe/add-new-vendor-to-directory/scribe-assets/step-02.png)

## Enter Vendor Information

Click the **Vendor Name** field.

![Vendor Name field](/help/scribe/add-new-vendor-to-directory/scribe-assets/step-03.png)

Type the vendor name.

Click the **Email** field.

![Vendor Email field](/help/scribe/add-new-vendor-to-directory/scribe-assets/step-05.png)

Type the vendor email address.

## Select Vendor Type

Click **Select vendor type...**.

![Vendor type dropdown](/help/scribe/add-new-vendor-to-directory/scribe-assets/step-07.png)

Choose the correct vendor type.

![Fabrication vendor type selected](/help/scribe/add-new-vendor-to-directory/scribe-assets/step-08.png)

## Add Business Details

Click the **Tax ID / EIN** field and enter the vendor tax ID if it is available.

![Tax ID / EIN field](/help/scribe/add-new-vendor-to-directory/scribe-assets/step-09.png)

Review the remaining business fields, including status, website, and payment terms.

![Vendor business details](/help/scribe/add-new-vendor-to-directory/scribe-assets/step-10.png)

## Link Contacts, If Needed

Use **Contacts** to link one or more existing contacts to the vendor during creation.

This step is optional. You can create the vendor without linked contacts, then link contacts later by opening the vendor profile, selecting **Edit Profile**, and adding contacts from the vendor edit form.

![Vendor contacts section](/help/scribe/add-new-vendor-to-directory/scribe-assets/step-11.png)

## Create the Vendor

Select **Create Vendor** after the required details are complete.

![Create Vendor action](/help/scribe/add-new-vendor-to-directory/scribe-assets/step-12.png)

## Quick Checklist

- Vendor name is entered.
- Email address is correct when available.
- Vendor type is selected.
- Tax ID / EIN is entered if available.
- Contacts are linked only if they are known during creation.
- **Create Vendor** is selected after review.
