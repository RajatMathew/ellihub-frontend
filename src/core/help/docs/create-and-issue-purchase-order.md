# Create and Issue a Purchase Order

Use this guide when a project needs a purchase order created, issued, and sent to the vendor. The screenshots show sample RFQs, cost codes, amounts, vendors, files, and email content only; use the actual project, vendor, cost breakdown, backup documents, and email recipients for the PO being issued.

## Start the Purchase Order

Open the project's **Purchase Orders** page.

![Project Purchase Orders page](/help/scribe/create-issue-purchase-order/scribe-assets/step-01.jpeg)

Select **Add PO**.

![Add PO action](/help/scribe/create-issue-purchase-order/scribe-assets/step-02.jpeg)

## Link the RFQ and Set PO Details

Open the RFQ selector.

![RFQ selector](/help/scribe/create-issue-purchase-order/scribe-assets/step-03.jpeg)

Choose the RFQ that should be linked to the purchase order.

![RFQ selected](/help/scribe/create-issue-purchase-order/scribe-assets/step-04.jpeg)

Set the PO date or expected delivery date fields required for the purchase order.

![PO date field](/help/scribe/create-issue-purchase-order/scribe-assets/step-05.jpeg)

## Build the Line Items

For each PO line, choose the correct cost code.

![Cost code selector](/help/scribe/create-issue-purchase-order/scribe-assets/step-06.jpeg)

![Cost code selected](/help/scribe/create-issue-purchase-order/scribe-assets/step-07.jpeg)

Repeat the same pattern for each additional cost line that belongs on the PO.

![Additional cost code selected](/help/scribe/create-issue-purchase-order/scribe-assets/step-11.jpeg)

Enter the item name, unit, quantity, rate, discount, shipping or handling fee, and tax values that apply to the purchase order. Review the subtotal, tax, and total before creating the PO.

![Line item totals](/help/scribe/create-issue-purchase-order/scribe-assets/step-25.jpeg)

## Attach Backup Documents

Use the backup document upload area to attach supporting files such as quotes, scope documents, vendor backup, or related project files.

![Backup document upload area](/help/scribe/create-issue-purchase-order/scribe-assets/step-26.jpeg)

Confirm the selected file appears before creating the PO.

![Backup document selected](/help/scribe/create-issue-purchase-order/scribe-assets/step-27.jpeg)

## Create the PO

Select **Create PO** after the RFQ, line items, totals, and backup documents have been reviewed.

![Create PO action](/help/scribe/create-issue-purchase-order/scribe-assets/step-28.jpeg)

Confirm the PO detail page opens with a PO number, draft status, linked RFQ, vendor information, financial summary, and key dates.

![Created PO detail page](/help/scribe/create-issue-purchase-order/scribe-assets/step-29.jpeg)

## Generate and Save the PDF

Select **Generate PDF** to create the purchase order PDF.

![Generate PDF action](/help/scribe/create-issue-purchase-order/scribe-assets/step-32.jpeg)

Select **Save to Files** when the generated PO PDF should be stored with the project files.

![Save PO PDF to files](/help/scribe/create-issue-purchase-order/scribe-assets/step-33.jpeg)

## Issue the PO

Select **Issue PO** from the PO detail page.

![Issue PO action](/help/scribe/create-issue-purchase-order/scribe-assets/step-35.jpeg)

Review the confirmation message and confirm **Issue PO**.

![Issue PO confirmation](/help/scribe/create-issue-purchase-order/scribe-assets/step-36.jpeg)

After issue, confirm the PO status changes to **Issued** and the issue date appears in the PO details.

![Issued PO status](/help/scribe/create-issue-purchase-order/scribe-assets/step-40.jpeg)

## Email the PO

Select **Email PO** from the issued PO detail page.

![Email PO action](/help/scribe/create-issue-purchase-order/scribe-assets/step-37.jpeg)

Continue from the PDF preview into the email step.

![Email PO continue action](/help/scribe/create-issue-purchase-order/scribe-assets/step-38.jpeg)

Review the sender, recipients, CC or BCC, subject, email body, and attachments. Select **Send PO Email** when the email is ready.

![Send PO Email action](/help/scribe/create-issue-purchase-order/scribe-assets/step-39.jpeg)

## Verify Project and PO Records

Return to the project overview when you need to verify how the issued PO affects project financials.

![Project financial verification](/help/scribe/create-issue-purchase-order/scribe-assets/step-48.jpeg)

Open **Purchase Orders** and confirm the PO appears with the correct vendor, status, amount, subtotal, discount, shipping, tax, and terms.

![Issued PO in list](/help/scribe/create-issue-purchase-order/scribe-assets/step-49.jpeg)

## Quick Checklist

- Correct project and Purchase Orders page are open.
- PO is linked to the correct RFQ.
- PO dates and vendor details are reviewed.
- Every line item has the correct cost code, quantity, unit, rate, and amount.
- Discount, shipping or handling, tax percentage, subtotal, and total are reviewed.
- Required backup documents are attached.
- PO detail page opens with a PO number after creation.
- PO PDF is generated and saved to project files when needed.
- PO is issued only after the draft details are correct.
- Email sender, recipients, body, and attachments are reviewed before sending.
- Project financials and Purchase Orders list reflect the issued PO.
