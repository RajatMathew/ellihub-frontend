# Create a New General Contractor

Use this workflow when a new general contractor needs to be added to the directory. General contractor profiles store company details, contractor type, payment terms, financial defaults, compliance details, and optional linked contacts.

## Start a New General Contractor

Navigate to **Directory > General Contractors**.

![General Contractors directory](/help/scribe/create-new-general-contractor/scribe-assets/step-01.png)

Select **Add GC**.

![Add GC action](/help/scribe/create-new-general-contractor/scribe-assets/step-02.png)

## Enter Company Information

Click the company name field.

![General contractor name field](/help/scribe/create-new-general-contractor/scribe-assets/step-03.png)

Type the general contractor name.

## Select GC Type and Payment Terms

Click **Select GC type...**.

![GC type dropdown](/help/scribe/create-new-general-contractor/scribe-assets/step-05.png)

Choose the correct GC type.

![GC type selected](/help/scribe/create-new-general-contractor/scribe-assets/step-06.png)

Review or change the payment terms.

![Payment terms field](/help/scribe/create-new-general-contractor/scribe-assets/step-07.png)

## Review Details

Review the main company details before moving through the remaining sections.

![General contractor details](/help/scribe/create-new-general-contractor/scribe-assets/step-08.png)

Continue through the form sections.

![Next form section](/help/scribe/create-new-general-contractor/scribe-assets/step-09.png)

Review or enter the financial default values if they apply.

![Financial default field](/help/scribe/create-new-general-contractor/scribe-assets/step-10.png)

Review the next financial default field.

![Additional financial default field](/help/scribe/create-new-general-contractor/scribe-assets/step-11.png)

## Link Contacts, If Needed

Use **GC Team Directory** to link one or more existing contacts to the general contractor during creation.

This step is optional. You can create the general contractor without linked contacts, then link contacts later by opening the GC profile, selecting **Edit Profile**, and adding contacts from the GC edit form.

![GC team directory section](/help/scribe/create-new-general-contractor/scribe-assets/step-12.png)

## Create the General Contractor

Select **Create GC** after the required details are complete.

![Create GC action](/help/scribe/create-new-general-contractor/scribe-assets/step-13.png)

## Quick Checklist

- General contractor name is entered.
- GC type is selected.
- Payment terms are correct.
- Financial defaults are reviewed.
- Contacts are linked only if they are known during creation.
- **Create GC** is selected after review.
