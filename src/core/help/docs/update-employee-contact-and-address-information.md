# Update Employee Contact and Address Information

Use this guide as a reference for where employee contact details can be edited. The example screenshots show sample changes only; use the actual email, phone, or address information that applies to the employee record you are updating.

## Find the Employee

Navigate to **HR > Employees**.

![Employees list](/help/scribe/update-employee-contact-and-address-information/scribe-assets/step-01.jpeg)

Select the employee profile that needs a contact information update.

![Employee selected](/help/scribe/update-employee-contact-and-address-information/scribe-assets/step-02.jpeg)

## Open the Profile Form

Select **Edit Profile** from the employee detail page.

![Edit Profile action](/help/scribe/update-employee-contact-and-address-information/scribe-assets/step-03.jpeg)

## Update Contact Details

Use the contact fields to update the information that has changed.

![Employee email field](/help/scribe/update-employee-contact-and-address-information/scribe-assets/step-04.jpeg)

The captured example shows an email edit. Replace the value with the correct email address for the employee.

![Updated email entered](/help/scribe/update-employee-contact-and-address-information/scribe-assets/step-05.jpeg)

The address field can be updated from the same profile form.

![Employee address field](/help/scribe/update-employee-contact-and-address-information/scribe-assets/step-06.jpeg)

Enter the correct address or location details for the employee.

![Updated address entered](/help/scribe/update-employee-contact-and-address-information/scribe-assets/step-07.jpeg)

## Save the Profile

Select **Save Profile** after reviewing the updated contact details.

![Save Profile action](/help/scribe/update-employee-contact-and-address-information/scribe-assets/step-08.jpeg)

After saving, confirm the profile reflects the new contact information.

![Updated address on profile](/help/scribe/update-employee-contact-and-address-information/scribe-assets/step-09.jpeg)

The profile contact details should now show the saved values.

![Updated email on profile](/help/scribe/update-employee-contact-and-address-information/scribe-assets/step-10.jpeg)

## Quick Checklist

- The selected profile is the correct employee.
- Only the fields that need to change are edited.
- The entered contact details are correct for that employee.
- **Save Profile** is selected after reviewing the changes.
- The profile shows the saved contact information after the update.
