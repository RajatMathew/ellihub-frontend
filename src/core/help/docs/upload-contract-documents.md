# Upload Contract Documents

Use this guide when prime contract documents need to be stored on a project. Uploaded contract documents appear in the project overview and are also available from **Files > Prime Contract**. The screenshots show sample file names only; upload the actual contract files for the project.

## Open Contract Documents

Open the project overview and find the **Contract Documents** card.

![Project overview contract documents card](/help/scribe/upload-contract-documents/scribe-assets/step-01.jpeg)

Select **Upload** in the **Contract Documents** card.

![Contract Documents upload action](/help/scribe/upload-contract-documents/scribe-assets/step-02.jpeg)

## Upload a Contract File

Drag a file into the upload area, or select the upload area to browse for a file.

![Upload drop zone](/help/scribe/upload-contract-documents/scribe-assets/step-03.jpeg)

Select **Upload** after the file is ready.

![Upload confirmation action](/help/scribe/upload-contract-documents/scribe-assets/step-04.jpeg)

Confirm the uploaded file appears in the **Contract Documents** card.

![Uploaded contract document](/help/scribe/upload-contract-documents/scribe-assets/step-05.jpeg)

## Upload Additional Contract Files

Use **Upload** again when more contract documents need to be added.

![Upload another contract document](/help/scribe/upload-contract-documents/scribe-assets/step-06.jpeg)

Add the next file from the upload area.

![Additional upload drop zone](/help/scribe/upload-contract-documents/scribe-assets/step-07.jpeg)

Select **Upload** to save the additional file.

![Additional upload confirmation](/help/scribe/upload-contract-documents/scribe-assets/step-08.jpeg)

Confirm the document count and file list update after upload.

![Updated contract document list](/help/scribe/upload-contract-documents/scribe-assets/step-09.jpeg)

## Pin Key Contract Documents

Pin the contract document that should stay visible from the project overview.

![Select contract document to pin](/help/scribe/upload-contract-documents/scribe-assets/step-10.jpeg)

Use **Pin** from the **Contract Documents** card when selecting documents from the Prime Contract folder.

![Pin contract document action](/help/scribe/upload-contract-documents/scribe-assets/step-11.jpeg)

Select the document that should appear on the project overview.

![Pin Prime Contracts selection](/help/scribe/upload-contract-documents/scribe-assets/step-12.jpeg)

Select **Pin** to confirm the pinned document selection.

![Confirm pinned contract document](/help/scribe/upload-contract-documents/scribe-assets/step-13.jpeg)

## Verify in Project Files

Open **Files** from the project sidebar.

![Project Files navigation](/help/scribe/upload-contract-documents/scribe-assets/step-14.jpeg)

Open **Prime Contract**.

![Prime Contract folder](/help/scribe/upload-contract-documents/scribe-assets/step-15.jpeg)

Confirm the uploaded contract document is available in the **Prime Contract** folder.

![Prime Contract folder contents](/help/scribe/upload-contract-documents/scribe-assets/step-16.jpeg)

## Quick Checklist

- The correct project is open before uploading.
- The uploaded files are actual project contract documents.
- Additional contract files are uploaded separately when needed.
- Key contract documents are pinned only when they should appear on the project overview.
- Uploaded documents are visible in **Files > Prime Contract**.
