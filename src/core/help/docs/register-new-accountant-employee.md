# Register a New Employee with Accountant Role

Use this workflow when adding a new employee who should have the **Accountant** role. The employee record can include contact details, employment role, optional system access, and emergency contact information.

## Start a New Employee

Navigate to **HR > Employees**.

![Employees list](/help/scribe/register-new-accountant-employee/scribe-assets/step-01.png)

Select **Add Employee**.

![Add Employee action](/help/scribe/register-new-accountant-employee/scribe-assets/step-02.png)

## Enter Employee Information

Click the employee name field.

![Employee name field](/help/scribe/register-new-accountant-employee/scribe-assets/step-03.png)

Type the employee name.

Click the email field.

![Employee email field](/help/scribe/register-new-accountant-employee/scribe-assets/step-05.png)

Type the employee email address.

Click the phone field.

![Employee phone field](/help/scribe/register-new-accountant-employee/scribe-assets/step-07.png)

Type the employee phone number.

Click the address field.

![Employee address field](/help/scribe/register-new-accountant-employee/scribe-assets/step-09.png)

Type the employee address.

## Assign the Accountant Role

Click **Select role...**.

![Employee role dropdown](/help/scribe/register-new-accountant-employee/scribe-assets/step-11.png)

Choose **Accountant**.

![Accountant role selected](/help/scribe/register-new-accountant-employee/scribe-assets/step-12.png)

## Set Up System Access, If Needed

Use the system access controls only if the employee should receive an auth-linked user account during employee creation.

![System access toggle](/help/scribe/register-new-accountant-employee/scribe-assets/step-13.png)

Choose the auth role for the linked user.

![Auth role option](/help/scribe/register-new-accountant-employee/scribe-assets/step-14.png)

Choose the **Accountant** auth role when the linked user should receive accounting access.

![Accountant auth role selected](/help/scribe/register-new-accountant-employee/scribe-assets/step-15.png)

Enter the temporary password or access value required by the form.

![System access field](/help/scribe/register-new-accountant-employee/scribe-assets/step-16.png)

This system access setup is optional during employee creation. You can create the employee record first, then link or configure the auth user later from the employee profile.

## Add Emergency Contact

Click the emergency contact name field.

![Emergency contact name field](/help/scribe/register-new-accountant-employee/scribe-assets/step-18.png)

Type the emergency contact name.

Click the emergency contact phone field.

![Emergency contact phone field](/help/scribe/register-new-accountant-employee/scribe-assets/step-20.png)

Type the emergency contact phone number.

Click the relationship field.

![Emergency contact relationship field](/help/scribe/register-new-accountant-employee/scribe-assets/step-22.png)

Type the emergency contact relationship.

## Create the Employee

Select **Create Employee** after the required information is complete.

![Create Employee action](/help/scribe/register-new-accountant-employee/scribe-assets/step-24.png)

After saving, confirm the employee profile and system access status.

![Employee system access status](/help/scribe/register-new-accountant-employee/scribe-assets/step-25.png)

## Quick Checklist

- Employee name, email, phone, and address are entered.
- **Accountant** is selected as the employee role.
- Auth user setup is completed only if access should be created immediately.
- Auth role setup can be skipped and completed later from the employee profile.
- Emergency contact details are entered.
- **Create Employee** is selected after review.
