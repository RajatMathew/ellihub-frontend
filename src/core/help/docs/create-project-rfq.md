# Create a New RFQ

Use this guide when a project needs a new request for quotation. The screenshots show sample RFQ text only; enter the actual bid deadline, track, title, scope, deliverables, quantities, and documents for the RFQ being created.

## Start a New RFQ

Open the project's **RFQs** page.

![Project RFQs page](/help/scribe/create-project-rfq/scribe-assets/step-01.jpeg)

Select **Add RFQ**.

![Add RFQ action](/help/scribe/create-project-rfq/scribe-assets/step-02.jpeg)

## Enter RFQ Details

Select the bid deadline.

![Bid deadline field](/help/scribe/create-project-rfq/scribe-assets/step-03.jpeg)

Open the **Track (Type)** selector.

![Track selector](/help/scribe/create-project-rfq/scribe-assets/step-04.jpeg)

Choose the track that matches the RFQ scope.

![Track selected](/help/scribe/create-project-rfq/scribe-assets/step-05.jpeg)

Enter a clear RFQ title.

![RFQ title field](/help/scribe/create-project-rfq/scribe-assets/step-06.jpeg)

Add the RFQ description or scope of work.

![RFQ description field](/help/scribe/create-project-rfq/scribe-assets/step-08.jpeg)

## Add Deliverables

Enter the item or deliverable name.

![Deliverable item field](/help/scribe/create-project-rfq/scribe-assets/step-10.jpeg)

Enter the quantity needed.

![Deliverable quantity field](/help/scribe/create-project-rfq/scribe-assets/step-12.jpeg)

Open the unit selector.

![Unit selector](/help/scribe/create-project-rfq/scribe-assets/step-14.jpeg)

Choose the unit that matches the deliverable.

![Unit selected](/help/scribe/create-project-rfq/scribe-assets/step-15.jpeg)

## Add Documents

Open **Documents** when supporting RFQ files need to be attached.

![Documents section](/help/scribe/create-project-rfq/scribe-assets/step-16.jpeg)

## Create and Confirm

Select **Create RFQ** after reviewing the RFQ details, deliverables, and documents.

![Create RFQ action](/help/scribe/create-project-rfq/scribe-assets/step-17.jpeg)

Confirm the RFQ detail page opens and the RFQ number is assigned. Newly created RFQs may still be in draft until they are published.

![Created RFQ detail page](/help/scribe/create-project-rfq/scribe-assets/step-18.jpeg)

## Quick Checklist

- The RFQ is being created in the correct project.
- Bid deadline is accurate.
- Track matches the RFQ scope.
- Title and description clearly explain the request.
- At least one deliverable is entered with quantity and unit.
- Supporting documents are added when needed.
- The created RFQ opens with an RFQ number.
