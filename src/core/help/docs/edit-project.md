# Editing a Project

Use this guide when an existing project needs a stage or status update. The screenshots show sample project data only; apply the stage and status that match the actual project workflow.

## Before You Edit

Open the project that needs to be updated.

![Project overview](/help/scribe/edit-project-stages-status/scribe-assets/step-01.jpeg)

Project edits fall into two groups:

- **Full project form edits** for changes that should be reviewed and saved from **Edit Project**.
- **Quick header edits** for stage changes that can be made directly from the project header.

## Full Project Form Edits

Use the full edit form when the change should be reviewed before saving, such as project status or a stage update made alongside other project details.

Select **Edit Project** from the project overview.

![Edit Project action](/help/scribe/edit-project-stages-status/scribe-assets/step-02.jpeg)

### Project Stage

Open the **Project Stage** selector.

![Project Stage selector](/help/scribe/edit-project-stages-status/scribe-assets/step-03.jpeg)

Choose the stage that matches the project's current workflow, such as **Submittals**.

![Project Stage option](/help/scribe/edit-project-stages-status/scribe-assets/step-04.jpeg)

Select **Save Changes** after reviewing the update.

![Save Changes action](/help/scribe/edit-project-stages-status/scribe-assets/step-05.jpeg)

Confirm the project overview shows the updated stage.

![Updated project stage](/help/scribe/edit-project-stages-status/scribe-assets/step-06.jpeg)

### Project Status

Use project status when the project should move between active and inactive tracking.

Open **Edit Project** when the active or inactive status needs to be updated.

![Edit Project from updated project](/help/scribe/edit-project-stages-status/scribe-assets/step-10.jpeg)

Open the **Project Status** selector.

![Project Status selector](/help/scribe/edit-project-stages-status/scribe-assets/step-11.jpeg)

Choose **Inactive** when the project should no longer appear as active work.

![Inactive status option](/help/scribe/edit-project-stages-status/scribe-assets/step-12.jpeg)

Use the date picker when an inactive date is required.

![Inactive date picker](/help/scribe/edit-project-stages-status/scribe-assets/step-13.jpeg)

Confirm the project status reflects the inactive setting.

![Inactive status selected](/help/scribe/edit-project-stages-status/scribe-assets/step-14.jpeg)

## Quick Header Stage Edits

Use the project header when only the stage needs to change and no other project details need review.

![Project header stage selector](/help/scribe/edit-project-stages-status/scribe-assets/step-07.jpeg)

Choose the next stage, such as **Buyout**, when the project is ready to move forward.

![Buyout stage option](/help/scribe/edit-project-stages-status/scribe-assets/step-08.jpeg)

Confirm the header shows the new stage.

![Buyout stage selected](/help/scribe/edit-project-stages-status/scribe-assets/step-09.jpeg)

## Quick Checklist

- The correct project is open before editing.
- Use **Edit Project** for status changes or stage changes that need review before saving.
- Use the project header for simple stage-only updates.
- Stage changes match the project's actual workflow.
- **Save Changes** is selected when editing through the full project form.
- Status is changed to **Inactive** only when the project should no longer be treated as active.
- The project overview or header reflects the final stage and status.
