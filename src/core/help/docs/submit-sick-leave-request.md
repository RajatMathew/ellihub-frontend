# Submit Sick Leave as an Admin

Use this guide as a reference for an admin recording a sick leave request for an employee in the HR Time Off area. The example screenshots show sample employee and request details only; choose the employee, dates, and request notes that match the actual leave request.

## Start a New Request

Navigate to **HR > Time Off**.

![Time Off list](/help/scribe/submit-sick-leave-request/scribe-assets/step-01.jpeg)

Select **New Request**.

![New Request action](/help/scribe/submit-sick-leave-request/scribe-assets/step-02.jpeg)

## Select the Employee

Open the employee selector.

![Employee selector](/help/scribe/submit-sick-leave-request/scribe-assets/step-03.jpeg)

Choose the employee whose sick leave needs to be recorded.

![Employee selected](/help/scribe/submit-sick-leave-request/scribe-assets/step-04.jpeg)

## Choose Leave Type

Open the leave type selector.

![Leave type selector](/help/scribe/submit-sick-leave-request/scribe-assets/step-05.jpeg)

Choose **Sick** as the leave type.

![Sick leave type selected](/help/scribe/submit-sick-leave-request/scribe-assets/step-06.jpeg)

## Add Request Context

Use the request notes field to add any context needed for review or record keeping.

![Request notes field](/help/scribe/submit-sick-leave-request/scribe-assets/step-07.jpeg)

Enter the relevant reason or note for the employee's sick leave request.

## Submit the Request

Select **Submit Request** after reviewing the form.

![Submit Request action](/help/scribe/submit-sick-leave-request/scribe-assets/step-09.jpeg)

After submitting, confirm the employee's sick leave request appears in the Time Off list.

![Submitted sick leave request](/help/scribe/submit-sick-leave-request/scribe-assets/step-10.jpeg)

## Quick Checklist

- The selected employee is correct.
- **Sick** is selected as the leave type.
- The request dates and notes match the employee's leave need.
- **Submit Request** is selected after reviewing the form.
- The submitted employee request appears in the Time Off list.
