# Connect a Fieldwire Project

Use this guide when an ElliHub Finance project needs to be linked to the matching Fieldwire project. The screenshots show sample project names and values only; select the Fieldwire project that matches the actual job.

## Open the Project

Open the project that needs the Fieldwire reference.

![Project overview](/help/scribe/connect-fieldwire-project/scribe-assets/step-01.jpeg)

Select **Edit Project**.

![Edit Project action](/help/scribe/connect-fieldwire-project/scribe-assets/step-03.jpeg)

## Select the Fieldwire Project

Open the **Fieldwire Project** selector.

![Fieldwire Project selector](/help/scribe/connect-fieldwire-project/scribe-assets/step-04.jpeg)

Search for the Fieldwire project by name or project number.

![Fieldwire project search](/help/scribe/connect-fieldwire-project/scribe-assets/step-05.jpeg)

Choose the matching Fieldwire project from the list.

![Fieldwire project selected](/help/scribe/connect-fieldwire-project/scribe-assets/step-06.jpeg)

## Save the Project

Select **Save Changes** after confirming the Fieldwire project is correct.

![Save Changes action](/help/scribe/connect-fieldwire-project/scribe-assets/step-07.jpeg)

## Verify the Synced Reference

Open the project financial area that contains change order totals.

![Project financial summary](/help/scribe/connect-fieldwire-project/scribe-assets/step-08.jpeg)

Open **Prime Change Orders**.

![Prime Change Orders tab](/help/scribe/connect-fieldwire-project/scribe-assets/step-09.jpeg)

Confirm the Fieldwire project reference and last sync details appear in Prime Change Orders.

![Fieldwire sync reference](/help/scribe/connect-fieldwire-project/scribe-assets/step-10.jpeg)

Review the **Reference** column to confirm change order references are available after the Fieldwire link is saved.

![Reference column](/help/scribe/connect-fieldwire-project/scribe-assets/step-11.jpeg)

## Quick Checklist

- The correct ElliHub Finance project is open.
- **Edit Project** is used to update the project record.
- The matching **Fieldwire Project** is selected.
- **Save Changes** is selected after updating the Fieldwire reference.
- Prime Change Orders shows the Fieldwire project and last sync details.
- The **Reference** column is available for synced Fieldwire references.
